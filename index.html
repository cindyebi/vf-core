In the right place?

This site is designed as a demonstration of all the components bundled with the Visual Framework 2.0 core.

You most likely don't need to clone the vf-core unless you're interested in improving the architecure of the system or contributing a global component for all users of the Visual Framework.

Most developers will want to build a site or design system that use the Visual Framework architecure and components, for those users we recommend:

  1. Reviewing the links below and components on this site; and
  2. Utilising the vf-eleventy templates.

Getting started

If you're new to the Visual Framework, here are some links to look at first.

  1. Understand the basics;
  2. Review the documentation;
  3. Browse the components below;
  4. Learn how to create new components or request a new component;
  5. Make your own site or design system; and
  6. Ask the community for help or open an issue.
