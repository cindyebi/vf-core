Development tools

First, see if you have the needed tools installed.

  1. Open a command line/terminal
  2. Enter node -v
  3. Enter gulp -v

If either of those steps return an error you need to install the corresponding software.

  • To install Node: go to the official downloads page
  • To install Gulp: Gulp requires Node and can be installed by: npm install gulpjs/gulp-cli -g

Installing the Visual Framework

You will need to install this repo onto your machine so that you can update, amend, and and delete components to the Visual Framework as required.

  1. Clone this repo
    • git clone https://github.com/visual-framework/vf-ebi.git
  2. Move to the directory
    • cd visual-framework-tooling-prototype
  3. You probably want the develop branch (or your new feature branch)
    • git checkout develop
  4. Install fractal You might need to use sudo.
    • npm i -g @frctl/fractal
  5. Download all the things
    • npm install
  6. Run a dev build and open in your browser
    • gulp dev
