Component versioning

Looking for notes on versioning of vf-core?

The versioning of components is self-contained and distinct from the version number of vf-core

Example: if vf-component@2.2.0 has a breaking change, it would be updated to vf-component@3.0.0. That new version number does not affect compatibility with with vf-core@2.x.
