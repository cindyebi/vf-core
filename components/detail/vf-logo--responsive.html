About

Installation and Implementation

This component is distributed with npm. After installing npm, you can install the vf-logo with this command.

$ npm install --save @visual-framework/vf-logo

The source files included are written in Sass(scss). You can point your Sass include-path at your node_modules directory and import it like this.

@import "@visual-framework/vf-logo/index.scss";

Make sure you import any requirements along with the modules.

Code example



<a href="/" class="vf-logo--responsive"></a>
