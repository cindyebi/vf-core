About

Installation and Implementation

This component is distributed with npm. After installing npm, you can install the vf-header with this command.

$ npm install --save @visual-framework/vf-header

The source files included are written in Sass(scss). You can point your Sass include-path at your node_modules directory and import it like this.

@import "@visual-framework/vf-header/index.scss";

Make sure you import any requirements along with the modules.

Code example



<header class="vf-header">

  {{render '@vf-global-header'}}

  {{render '@vf-masthead'}}

  {{render '@vf-navigation--primary'}}

  {{render '@vf-navigation--secondary'}}

</header>
