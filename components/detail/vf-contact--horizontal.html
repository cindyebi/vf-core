About

Installation and Implementation

This component is distributed with npm. After installing npm, you can install the vf-contact with this command.

$ npm install --save @visual-framework/vf-contact

The source files included are written in Sass(scss). You can point your Sass include-path at your node_modules directory and import it like this.

@import "@visual-framework/vf-contact/index.scss";

Make sure you import any requirements along with the modules.

Code example



<address class="vf-contact vf-contact--horizontal">

  <p class="vf-contact__team">EMBL-EBI Press Office</p>
  <p class="vf-contact__name">Laura Howes</p>

  <p class="vf-contact__job-title">Social media manager</p>

  <a href="mailto:digital-comms@embl.org" class="vf-contact__email">digital-comms@embl.org</a>

  <a href="tel:01234567890" class="vf-contact__phone-number">01234 567 890</a>
  <a href="tel:09876543210" class="vf-contact__phone-number vf-contact__phone-number--mobile">09876 543 210</a>

  <img class="vf-contact__image vf-contact__image--circle" src="http://dev-embl-communications.pantheonsite.io/wp-content/uploads/2018/07/female-head-300x300.jpg" alt="image of the contact">

</address>

// vf-contact

.vf-contact {
  --vf-contact__margin--bottom: 8px;

  align-items: flex-start;
  display: flex;
  flex-direction: column;
  font-style: normal; // the address element italicises any typography
  margin: 0 0 24px 0;

  /* we don't want the last phone number or email to have any bottom margin */
  > a:last-child {
    margin-bottom: 0;
  }

  /* In the source order the image is at the bottom. For the layout
   * the image needs to be between the contact details and links. So we set
   * the order of these with CSS, targetting the elements, rather than classes
   */
  p {
    order: 1;
  }

  img {
    order: 2;
  }

  a {
    order: 3;
  }

  /* We are using CSS custom properties, for IE11 we need a generic fallback */
  > * {
    margin: 0 0 12px 0;
  }
}

/* The horizontal contact component has an image on the left and the text on
 * on the right.
 */

.vf-contact--horizontal {
  display: grid;
  grid-column-gap: 32px;
  grid-template-columns: auto 1fr;
  grid-template-rows: repeat(6, min-content);

  > * {
    // we want all details of the contact to be in the 2nd column of the grid.
    grid-column: 2;
  }

  /* As the text is on the right, having a bottom margin on the last phone-number or a element causes the
  * avatar to look misaligned, removing the maring-bottom on it fixes this.
  */
  a:last-of-type {
    margin-bottom: 0;
  }

  .vf-contact__email {
    --vf-contact__margin--bottom: 18px;
  }

    align-self: center;
    grid-column: 1;
    grid-row: 1 / -1;
    margin-bottom: 0;
    max-width: 170px;
  }
