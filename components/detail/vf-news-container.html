Installation and Implementation

This component is distributed with npm. After installing npm, you can install the vf-news-container with this command.

$ npm install --save @visual-framework/vf-news-container

The source files included are written in Sass(scss). You can point your Sass include-path at your node_modules directory and import it like this.

@import "@visual-framework/vf-news-container/index.scss";

Make sure you import any requirements along with the modules.

<section class="vf-news-container | embl-grid embl-grid--has-sidebar">
