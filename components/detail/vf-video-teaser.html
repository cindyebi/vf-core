About

Installation and Implementation

This component is distributed with npm. After installing npm, you can install the vf-video-teaser with this command.

$ npm install --save @visual-framework/vf-video-teaser

The source files included are written in Sass(scss). You can point your Sass include-path at your node_modules directory and import it like this.

@import "@visual-framework/vf-video-teaser/index.scss";

Make sure you import any requirements along with the modules.

Code example



<div class="vf-video-teaser">
    <h3 class="vf-video-teaser__title">More on our YouTube channel</h3>
    <a href="#"><img class="vf-video-teaser__image" src="../../assets/vf-video-teaser/assets/video-teaser.png" alt=""></a>
    <a href="#" class="vf-video-teaser__link vf-link">New theory deepens understanding of Turing patterns</a>
</div>
