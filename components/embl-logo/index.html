EMBL Logo Component

About

Install

This component is distributed with npm. After installing npm, you can install the embl-logo with this command.

$ yarn add --dev @visual-framework/embl-logo

The source files included are written in Sass(scss). You can point your Sass include-path at your node_modules directory and import it like this.

@import "@visual-framework/embl-logo/index.scss";

Make sure you import Sass requirements along with the modules.

Variants

Variant: Default live Type: embl-element

Nunjucks demo

{% render '@embl-logo--default', {
  "component-type": "embl-element",
  "logo_text": "European Molecular Biology Laboratory Home",
  "logo_href": "/"
} %}

Sample output

European Molecular Biology Laboratory Home

Sample output as HTML

<a href="/" class="embl-logo">
  <span class="vf-u-sr-only" for="text">European Molecular Biology Laboratory Home</span>
</a>

Variant: Extreme live Type: embl-element

Nunjucks demo

{% render '@embl-logo--extreme', {
  "component-type": "embl-element",
  "logo_text": "European Molecular Biology Laboratory Home",
  "logo_href": "/",
  "classname": "embl-logo--extreme"
} %}

Sample output

European Molecular Biology Laboratory Home

Sample output as HTML

<a href="/" class="embl-logo embl-logo--extreme">
  <span class="vf-u-sr-only" for="text">European Molecular Biology Laboratory Home</span>
</a>

Variant: Responsive live Type: embl-element

Nunjucks demo

{% render '@embl-logo--responsive', {
  "component-type": "embl-element",
  "logo_text": "European Molecular Biology Laboratory Home",
  "logo_href": "/",
  "deprecated_text": "'@embl-logo--responsive' is deprecated, use '@embl-logo'"
} %}

Sample output

European Molecular Biology Laboratory Home

Sample output as HTML

<a href="/" class="embl-logo">
  <span class="vf-u-sr-only" for="text">European Molecular Biology Laboratory Home</span>
</a>
<!-- '@embl-logo--responsive' is deprecated, use '@embl-logo' -->
