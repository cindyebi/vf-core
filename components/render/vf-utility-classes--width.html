Width

Use the utility clasees .vf-u-width__20 to set responive width in 5 percentage increments.

.vf-u-width__5

.vf-u-width__10

.vf-u-width__15

.vf-u-width__20

.vf-u-width__30

.vf-u-width__40

And so on up to 100.

Responsive

You can also make these conditional on screen size with --xs, --sm, --md, --lg or --xl.

.vf-u-width__20--sm

.vf-u-width__20--xl
